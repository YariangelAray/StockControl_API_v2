📘 Endpoints disponibles

Estos son los endpoints REST disponibles en /stockcontrol_api:

  • /usuarios
    • GET /usuarios Lista todos los usuarios
    • GET /usuarios/:id Consulta un usuario por ID
    • POST /usuarios Crea un nuevo usuario
    • PUT /usuarios/:id Actualiza un usuario existente
    • PATCH /usuarios/:id Actualiza parcialmente un usuario
    • DELETE /usuarios/:id Elimina un usuario por ID
    • PUT /usuarios/:id/contrasena Actualiza la contraseña validando la anterior
  • /roles-usuarios
    • GET /roles-usuarios Lista todas las relaciones rol-usuario
    • GET /roles-usuarios/:id Consulta una relación específica
    • POST /roles-usuarios Asigna un rol a un usuario
    • PUT /roles-usuarios/:id Actualiza una relación existente
    • PATCH /roles-usuarios/:id Actualiza parcialmente una relación
    • DELETE /roles-usuarios/:id Elimina una relación rol-usuario
  • /roles
